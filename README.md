# 🎓 Student Grade Management System

> Java를 활용한 콘솔 기반 학생 성적 관리 프로그램
> Git을 이용한 팀 협업 프로젝트

---

## 📖 프로젝트 소개

학생의 성적 정보를 효율적으로 관리하기 위한 프로그램입니다.

사용자는 학생을 등록하고, 성적을 입력하며, 평균 점수 및 최고 점수 학생을 조회할 수 있습니다.

본 프로젝트는 Git과 GitHub를 활용한 협업 능력 향상을 목표로 진행되었습니다.

---

## ✨ 주요 기능

| 기능       | 설명            |
| -------- | ------------- |
| 학생 등록    | 학생 이름 및 점수 입력 |
| 학생 조회    | 등록된 학생 목록 출력  |
| 평균 점수 계산 | 전체 학생 평균 계산   |
| 최고 점수 조회 | 최고 점수 학생 출력   |
| 프로그램 종료  | 안전하게 프로그램 종료  |

---

## 📋 클래스 설명

### Student.java

학생 정보를 저장하는 클래스

```java
private String name;
private int score;
```

---

### StudentManager.java

학생 목록을 관리하는 클래스

기능

* 학생 등록
* 학생 조회
* 평균 계산
* 최고 점수 조회

---

### Main.java

프로그램 실행 및 메뉴 관리

```java
while(true){
    // 메뉴 출력
}
```
